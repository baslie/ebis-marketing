## 0. Логотип

**Файл:** `landing/src/assets/ebis-logo.svg` (векторный, 1357×446)

### Структура логотипа
- **Иконка** — шестиугольник с вписанной буквой «Э»
- **Текст** — «ЭБИС» (часть SVG, не шрифт)

### Цвета логотипа
| Элемент | Цвет | HEX |
|---------|------|-----|
| Шестиугольник (заливка) | Оранжевый | `#FE6601` |
| Буква «Э» (градиент) | Золотой → Оранжевый | `#FFA800` → `#FE6601` |
| Текст «ЭБИС» | Чёрный | `#000000` |

> **Важно:** цвет логотипа `#FE6601` отличается от CSS-переменных сайта (`--primary-color: #F36F2E`, CTA: `#F36000`). Для лендинга рекомендуется использовать SVG-логотип as-is и выбрать `#FE6601` или `#F36000` как единый primary-цвет.

### Использование
- **Хедер** — логотип слева, высота ~40px, горизонтальная ориентация
- **Футер** — логотип меньшего размера, ~24px высота
- **Favicon** — только иконка (шестиугольник), без текста

## 6. Сетка и брейкпоинты (Tailwind CSS)

12-колоночная сетка с адаптивными брейкпоинтами Tailwind.

### Брейкпоинты
| Prefix | Min-width | Container max-width | Назначение |
|--------|-----------|---------------------|------------|
| `sm` | 640px | 640px | Мобильные (landscape) |
| `md` | 768px | 768px | Планшеты |
| `lg` | 1024px | 1024px | Ноутбуки |
| `xl` | 1280px | 1280px | Десктопы |
| `2xl` | 1536px | 1536px | Широкие экраны |

### Типовые раскладки (grid-cols-12)
| Раскладка | Колонки | Использование |
|-----------|---------|---------------|
| 12 | 1 &times; 12 | Полная ширина (hero, CTA-секции) |
| 6 + 6 | 2 равные | Текст + изображение |
| 4 + 4 + 4 | 3 равные | Карточки преимуществ, контактная полоса |
| 3 + 3 + 3 + 3 | 4 равные | Статистика, иконки |
| 8 + 4 | Контент + сайдбар | Каталог, текст + форма |
| 3 + 9 | Сайдбар + контент | Навигация + основной контент |

## 8. Компоненты (паттерны)

### Хедер (sticky)
- Белый фон, логотип слева, навигация по центру, кнопка телефона справа
- Навигация: Montserrat 16px / 300, чёрный текст
- Телефон: оранжевая pill-кнопка (`border-radius: 30px`), белый текст, Montserrat 14px / 600

### CTA-кнопки
- Фон: `#F36000` (ярко-оранжевый)
- Текст: белый, Montserrat 14px, semibold (600)
- Скругление: 30px (pill-форма)
- Hover: `#E55A1F` (тёмнее)
- Без теней

### Контактная полоса
- 3 карточки в ряд с серым фоном (`#F5F5F5`)
- Центральная карточка — оранжевый фон с CTA "Рассчитать стоимость"
- Иконки + текст

### Статистика ("Почему покупают у нас")
- Крупные числа: оранжевый цвет, ~48px, italic
- Подписи: чёрный, uppercase, 14–16px

### Таблица сравнения
- Чередующиеся строки: белый / `#F5F5F5`
- Заголовки столбцов: bold
- Чистая без бордеров таблица

### Форма заявки
- Инпуты: Montserrat 16px, `border: 1px solid black`, высота 60px, `padding: 0 20px`
- Серый фон секции
- Кнопка отправки: оранжевая pill, `border-radius: 50px`

### FAQ-аккордеон
- Серый фон секции (`#F5F5F5`)
- Разделители: тонкие линии
- Иконка "+" справа

### Футер
- Белый фон
- Навигация ссылками
- Логотип ЭБИС

## 10. Общие наблюдения о дизайне

- **Стиль**: Чистый, профессиональный, промышленный. Минимум декора.
- **Цветовая схема**: Монохром (чёрный + серые) + единственный акцент — оранжевый.
- **Типографика**: Montserrat во всех вариантах (300–700). Заголовки крупные (47–48px). Контраст достигается через weight, а не через цвет.
- **Кнопки**: Исключительно pill-форма (30–50px radius). Всегда оранжевые с белым текстом.
- **Фоны секций**: Чередование белого и светло-серого (`#F5F5F5`).
- **Формы**: Минималистичные инпуты с нижним подчёркиванием или тонким бордером.
- **Иконки**: Линейные, оранжевые на белом фоне (или белые на оранжевом).
- **Изображения**: Крупные фото домов с металлочерепицей, lifestyle-фото.
- **Виджеты**: Telegram-кнопка (голубая, правый нижний угол), чат "Есть вопросы?".
